# 지역행사 알리미 - Android

## 🖥️ Project info 


스마트폰 어플리케이션을 활용하여 내가 선택한 각 지역의 행사 정보(뮤지컬,연극,음악회,콘서트)를 실시간으로 확인할 수 있고 현 시점에 참여할 수 있는 공연 정보와 앞으로 예정된 정보를 손쉽게 확인할 수 있는 서비스를 제공하는 모바일 앱 서비스
